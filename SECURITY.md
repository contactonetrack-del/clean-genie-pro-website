# Security Policy

## Supported Versions

The following versions of Clean Genie Pro receive security updates and patches:

| Version | Supported | Notes |
|---------|-----------|-------|
| 2.x     | ✅ Yes    | Current stable release |
| 1.x     | ❌ No     | Outdated, please upgrade |
| < 1.0   | ❌ No     | Legacy versions |

## Reporting a Vulnerability

If you discover a security vulnerability in Clean Genie Pro, please report it responsibly instead of using the public issue tracker.

### How to Report

**Email:** contactonetrack@gmail.com

Please include the following information in your report:

- Description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Affected version(s) and system environment
- Any proof-of-concept or code that demonstrates the issue
- Your contact information for follow-up

### GitHub Security Advisory

Alternatively, you can open a private security advisory:
1. Go to **Security** tab → **Advisories**
2. Click **Report a vulnerability**
3. Provide details and submit

## Our Commitment

We take security seriously and commit to:

- **Initial Response:** Within 3-5 business days of your report
- **Updates:** Regular status updates on remediation efforts
- **Timeline:** We aim to release a fix within 30 days of confirmation
- **Disclosure:** Public disclosure only after a fix is available and released
- **Credit:** Recognition of your responsible disclosure (if desired)

## Security Best Practices

When using Clean Genie Pro:

- Keep the application updated to the latest version
- Run with appropriate user permissions (not always as administrator)
- Regularly review system changes made by the application
- Report any unexpected behavior immediately

## FAQ

**Q: Do you offer a bug bounty program?**  
A: Not currently, but we deeply appreciate responsible disclosures.

**Q: How do I know if my report was received?**  
A: We'll send an acknowledgment email within 2 business days.

**Q: Can I disclose this publicly if I don't hear back?**  
A: Please give us at least 90 days before public disclosure.
